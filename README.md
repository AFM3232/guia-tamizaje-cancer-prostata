# Guía interactiva — Tamizaje de Cáncer de Próstata (USPSTF 2018)

Guía de estudio **interactiva y responsive** (móvil/tablet/escritorio) para preparar el debate informado sobre el tamizaje con PSA, basada en la declaración de recomendación del **US Preventive Services Task Force**.

> Fuente: US Preventive Services Task Force. *Screening for Prostate Cancer: USPSTF Recommendation Statement.* JAMA. 2018;319(18):1901-1913. doi:10.1001/jama.2018.3710.

## Qué incluye
- Plan de estudio paso a paso.
- Recomendaciones núcleo: **C** (55–69 años) y **D** (≥70 años).
- Sistema de grados USPSTF (A–B–C–D–I) y niveles de certeza.
- PSA y desempeño diagnóstico (sensibilidad, especificidad, FP/FN).
- Factores de riesgo, beneficios y daños (sobrediagnóstico, falsos positivos).
- Tratamientos y sus daños.
- Los 3 ensayos clave: **PLCO, ERSPC, CAP**.
- Poblaciones especiales (afroamericanos, antecedente familiar).
- Cambio 2012 → 2018 y recomendaciones de otras organizaciones.
- **Kit de argumentos para el debate** (fuertes a favor / en contra + débiles con su contraataque).
- Tabla de "los 1000 hombres", glosario, mnemotecnias y **autoevaluación con 10 preguntas**.

Todo el contenido es una elaboración educativa propia con cifras tomadas del artículo citado.

## Uso
Abre la página publicada (GitHub Pages) o el archivo `index.html` en cualquier navegador. Es totalmente offline.
